# Metropolis Bites - 都会の隠れ家グルメガイド

## 概要
都会の喧騒を忘れさせる、隠れ家的なレストランやカフェを紹介するグルメガイドサイトです。
既存のテンプレートをベースに、デザインと機能を刷新し、ユーザーが「自分だけの特別なお店」を見つけられるプラットフォームを目指しました。

## 特徴・コンセプト
*   **ターゲット層**: 20代〜40代の、落ち着いた空間で食事を楽しみたい都市生活者。
*   **デザインテーマ**: 「都会の夜と温かい灯り」をイメージし、ネイビー（#1e3a8a）とオレンジ（#d97706）を基調とした配色で、高級感と親しみやすさを両立させました。

## 変更点・工夫した点（発表用メモ）
### 1. レイアウトとBootstrapの活用
*   **ヒーローセクション (index.html)**:
    *   ファーストビューでサイトの世界観を伝えるため、グラデーション背景と画像を組み合わせたインパクトのあるデザインに変更しました。
    *   Bootstrapのグリッドシステム (col-lg-6) を使い、PCでは左右2カラム、スマホでは縦積みになるレスポンシブ対応を行いました。
*   **店舗詳細ページのカルーセル (shop-detail.html)**:
    *   BootstrapのCarouselコンポーネントを使用し、店舗の雰囲気が伝わる複数の写真をスライド表示できるようにしました。
*   **検索フィルター (shop-list.html)**:
    *   PC表示時に追従するサイドバー (sticky-top) を実装し、スクロールしても検索条件をすぐに変更できるようにしました。

### 2. オリジナル要素の追加
*   **Q&A機能 (gourmet-qa.html)**:
    *   ユーザー同士の交流を促すため、Q&A一覧ページを新規作成しました。
    *   解決済み・未解決のステータスバッジを表示し、視認性を高めています。
*   **食べ歩きログ (ating-log.html)**:
    *   ユーザーが訪れたお店を記録できるマイページ機能を想定したレイアウトを作成しました。
    *   グリッドレイアウトで写真を大きく見せるデザインにしています。

### 3. コードの品質・保守性
*   **CSSのリファクタリング**:
    *   各HTMLファイルに散らばっていた <style> タグを廃止し、全てのスタイルを css/style.css に集約しました。これにより、サイト全体の色やフォントの一括変更が容易になりました。
    *   CSS変数（:root）を使用し、テーマカラー（--primary-color, --secondary-color）を一元管理しています。
*   **コンポーネントの共通化**:
    *   ボタン (.btn-primary-custom) やカード (.shop-card) などの共通クラスを作成し、デザインの統一感を保ちつつ、コードの重複を減らしました。

## ページ構成
1.  **ホーム** (index.html): サイトの顔となるトップページ。
2.  **店舗一覧** (shop-list.html): 条件検索機能付きの一覧ページ。
3.  **店舗詳細** (shop-detail.html): 店舗情報、地図、口コミを表示。
4.  **グルメコラム** (gourmet-guide.html): 特集記事一覧ページ。
5.  **Q&A** (gourmet-qa.html): ユーザー参加型のQ&Aページ。
6.  **食べ歩きログ** (ating-log.html): ユーザーの記録ページ。
7.  **ログイン** (login.html): 認証ページ。
8.  **新規登録** (signup.html): 新規登録ページ。
9.  **口コミ投稿** (eview-form.html): フォーム入力ページ。
