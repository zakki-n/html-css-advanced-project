# Metropolis Bites - 都会の隠れ家グルメガイド

## 概要
都会の喧騒を忘れさせる、隠れ家的なレストランやカフェを紹介するグルメガイドサイトです。
既存のテンプレートをベースに、デザインと機能を刷新しました。

## 特徴
- **厳選された店舗情報**: 編集部が厳選した「隠れ家」スポットを紹介。
- **Q&A機能**: ユーザー同士でグルメに関する疑問を解決。
- **食べ歩きログ**: 自分のグルメ体験を記録。
- **レスポンシブデザイン**: スマートフォンやタブレットでも快適に閲覧可能。

## 使用技術
- HTML5
- CSS3 (Bootstrap 5.3, Custom CSS)
- JavaScript (Bootstrap Bundle, Custom JS)

## ページ構成
1. **ホーム** (index.html): サイトのトップページ。おすすめ店舗や特徴を紹介。
2. **店舗一覧** (shop-list.html): 条件に合わせて店舗を検索。
3. **店舗詳細** (shop-detail.html): 店舗の詳しい情報や口コミを表示。
4. **グルメコラム** (gourmet-guide.html): グルメに関する特集記事。
5. **Q&A** (gourmet-qa.html): グルメに関する質問と回答。
6. **食べ歩きログ** (ating-log.html): ユーザーの食べ歩き記録一覧。
7. **ログイン** (login.html): ユーザーログインページ。
8. **新規登録** (signup.html): 新規アカウント登録ページ。
9. **口コミ投稿** (eview-form.html): 店舗への口コミ投稿フォーム。

## 工夫した点
- **デザインの統一感**: ネイビーとオレンジを基調とした、落ち着きつつも温かみのある配色にしました。
- **コンポーネントの活用**: Bootstrapのカード、ナビゲーションバー、モーダルなどを効果的に活用し、見やすく使いやすいレイアウトを実現しました。
- **コードのリファクタリング**: 共通のCSSをstyle.cssに集約し、各HTMLファイルの記述をシンプルに保ちました。
- **オリジナル要素**: 「Q&A」や「食べ歩きログ」など、ユーザー参加型のコンテンツを想定したページを追加しました。
